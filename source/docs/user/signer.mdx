# Signer Guide

![Dimension Signer Logo](https://docs.dscc.community/assets/DimensionNetwork/dimension-signer-logo.jpg)

DimensionLabs Signer acts as your DSCC wallet, keeping your accounts and letting you sign deploys on the Dimension Network to perform actions like staking, unstaking or sending your tokens to another person or an exchange account.

You can create, store and use one or more DSCC accounts on your Signer wallet. All accounts are stored safely in a vault, which is protected by a password you will set when you create it.

## New User (No Accounts)

### Download Signer

Using Chrome or a Chromium-based browser like Brave, visit the Chrome Web Store, then download and install the [DimensionLabs Signer extension](https://chrome.google.com/webstore/detail/dimensionlabs-signer/djhndpllfiibmcdbnmaaahkhchcoijce).
